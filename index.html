Requêtes SQL avancées du projet.

Top contenus notés

SELECT c.titre, c.type, ROUND(AVG(n.note),2) AS note_moy, COUNT(*) AS nb_avis FROM CONTENU c JOIN NOTATION n ON c.id_contenu = n.id_contenu GROUP BY c.id_contenu, c.titre, c.type ORDER BY note_moy DESC LIMIT 3;
→ Breaking Bad (4.9) · Inception (4.8) · Interstellar (4.7)

Abonnés Premium ayant visionné >5h ce mois

SELECT u.nom, u.email, SUM(v.duree_vue) AS total_min FROM UTILISATEUR u JOIN ABONNEMENT a ON u.id_abonnement = a.id_abonnement JOIN VISIONNAGE v ON u.id_user = v.id_user WHERE a.type_plan = 'Premium' AND DATE_TRUNC('month', v.date_vis) = DATE_TRUNC('month', NOW()) GROUP BY u.id_user, u.nom, u.email HAVING SUM(v.duree_vue) > 300 ORDER BY total_min DESC;
→ Amîn (420 min) · Sofia (380 min)

Revenus mensuels par plan

SELECT a.type_plan, COUNT(u.id_user) AS nb, a.prix_mensuel, COUNT(u.id_user) * a.prix_mensuel AS revenu FROM ABONNEMENT a LEFT JOIN UTILISATEUR u ON u.id_abonnement = a.id_abonnement GROUP BY a.id_abonnement, a.type_plan, a.prix_mensuel ORDER BY revenu DESC;
→ Premium : 2×17.99€ = 35.98€ · Standard : 10.99€ · Basique : 5.99€
